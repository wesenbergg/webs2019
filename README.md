# Metropolia web-kurssi syksy 2019

Jakson seuraa väljästi materiaalia, joka löytyy osoitteesta https://github.com/covcom/205CDE. Käsiteltävät kappaleet ovat 1 - 16. Eli MVC on viimeinen,  joka käydään läpi.
